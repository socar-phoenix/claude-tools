# phoenix-flow

반복 작업을 자동화하는 Claude Code 플러그인. 실제 작업 로그에서 반복 빈도가 가장 큰 흐름을 도구로 묶는다.

> **상태**: 0.1.0 — 빈 뼈대. 아래 기능은 각자 별도 사이클로 하나씩 채운다.

## 로드맵

1. **API 테스트 러너** — 반복되는 `curl` + 응답 확인 흐름을 케이스 정의 한 번으로 줄인다. (skill + MCP)
2. **cd·gh 재시도 자동 교정** — 자주 가는 경로 점프 + `gh` 명령 실패 진단·재시도. (hook, 무개입)
3. **조회 헬퍼** — 반복되는 파일 읽기→grep 흐름을 정해진 조회로. (skill)

## 디렉터리

| 경로 | 용도 |
|------|------|
| `.claude-plugin/plugin.json` | 플러그인 매니페스트 |
| `skills/` | 스킬 (러너·조회 헬퍼 자리) |
| `hooks/` | hook (cd·gh 재시도 교정 자리) |
| `scripts/` | 실행 스크립트 자리 |

각 플러그인은 self-contained — 실행 컴포넌트는 전부 이 폴더 안에 둔다(source root 밖 참조 불가).
